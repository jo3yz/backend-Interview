## 为什么要握手
因为保证可靠的全双工通信，需要交换序列号，大概像这样
```
Client ---------- Server
|                   |
|--------syn------->|
|<------syn/ack-----|
|--------ack------->|
```

## 为什么是三次
三次就够了，因为没有“半打开”状态，由于通信不可靠的问题，握手再许多次也意义不大。
